\# 📊 Hospitality Dashboard Analysis



\## 📌 Project Overview



This project focuses on analyzing hotel performance data to uncover key business insights related to revenue, occupancy, and customer behavior. Interactive dashboards were built to support data-driven decision-making.



\---



\## 🎯 Objectives



\* Analyze hotel performance metrics

\* Identify trends in revenue and bookings

\* Understand customer behavior and preferences

\* Improve pricing and operational strategies



\---



\## 🛠️ Tools \& Technologies Used



\* \*\*Power BI\*\* – Dashboard creation \& visualization

\* \*\*Tableau\*\* – Interactive data visualization

\* \*\*Excel\*\* – Data cleaning \& preprocessing

\* \*\*SQL\*\* – Data extraction and transformation



\---



\## 📈 Key Metrics (KPIs)



\* \*\*Revenue (RevPAR)\*\*

\* \*\*Average Daily Rate (ADR)\*\*

\* \*\*Occupancy Rate\*\*



\---



\## 🔍 Key Insights



\* Identified peak booking periods and seasonal trends

\* Analyzed customer preferences and booking patterns

\* Helped improve pricing strategy using data insights

\* Enhanced operational efficiency through KPI tracking



\---



\## 📂 Project Files



\* `Excel Dashboard.xlsx` – Data preprocessing \& analysis

\* `Hospital.sql` – Database queries

\* `PowerBI Dashboard.pbix` – Power BI dashboard

\* `Tableau Dashboard.twbx` – Tableau dashboard



\---



\## 🚀 How to Use



1\. Open Power BI file to view interactive dashboard

2\. Use Tableau workbook for additional visual insights

3\. Refer SQL file for data queries

4\. Excel file contains cleaned dataset



\---



\## 💡 Conclusion



This project demonstrates how data analytics can improve business decisions in the hospitality industry by leveraging visualization tools and structured data analysis.
